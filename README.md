# Challenge AluraStore Latam 📊

Este proyecto es parte del desafío de análisis de datos de **Alura Latam**. Consiste en el procesamiento, limpieza y visualización de datos de una tienda ficticia (AluraStore) para extraer insights sobre el comportamiento de ventas.

## 🚀 Objetivo del Proyecto
El objetivo principal es aplicar técnicas de análisis de datos utilizando **Python** y **Pandas** para resolver problemas de negocio, tales como:
* Identificar los productos más vendidos.
* Analizar la evolución de las ventas en el tiempo.
* Segmentación básica de clientes o categorías.

## 🛠️ Herramientas Utilizadas
* **Google Colab**: Entorno de desarrollo para la ejecución del notebook.
* **Python**: Lenguaje principal.
* **Pandas**: Manipulación y limpieza de datos.
* **Matplotlib**: Visualización de datos (gráficos).

## 📁 Estructura del Repositorio
* `Challenge_AluraStoreLatam_fromerob.ipynb`: Notebook principal con todo el análisis, código y conclusiones.
* `README.md`: Descripción del proyecto.
